# 通过 ssh 连接 Linux 系统

本教程上半部分以 SecureCRSecureFXPortable 为例

下半部分 xshell 仅供简单参考，我个人使用，感兴趣的可以切换过来

# SecureCRSecureFXPortable 教程

# ssh 连接方法：

1. 关于 SecureCRSecureFXPortable 的程序组成介绍，注意：win10 以上系统需要兼容性设置才能运行！

2. 通过 ip a 命令查看 ip 地址（ssh 连接 host 地址）

3. 新建一个 ssh 连接

4. 填入基本 ssh 连接信息

5. 继续相关配置

6. 接受秘钥

7.ssh 成功登入系统（此处使用插件 “neofetch” 展示了系统基本信息）

# 配套 ftp 工具使用方法：

1. 在 ssh 连接配置完毕后，直接打开 ftp 工具，选择刚才的服务器连接即可

2. 一样，接受秘钥

3. 登入 ftp 服务器，左边是本地计算机，右边是服务器端，鼠标右键可选择操作
