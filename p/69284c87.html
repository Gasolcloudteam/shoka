# 部分问题

# 1）关于没有收到发货邮件的问题

答：确保你收到了回执邮件，然后等待两天左右，如果超过一周还是没有收到发货邮件，再多申请几次，这个原因不明，有可能是审核没通过，已经有小伙伴反馈收到发货邮件或者贴纸了，所以没收到的按照上面的方法再试试吧。（貌似 QQ 邮箱会吞回执邮件，但 up 本地测试是正常的）

# 2）申请提交不上去

答：实际上是谷歌的人机验证平台（reCaptcha）加载不出来，不会科学上网的小白，去下载 Steam++（现在叫 Watt Toolkit），勾选 “国外验证码平台” 然后点击加速就可以了。

直达链接：点击跳转

# 3）很老很偏门或者笔记本，E3E5，ESQS 的型号可以吗

答：只要能过验证都行，会给你发对应系列的贴纸

# 4）需要付邮费吗

答：不需要，intel 还会对你说 thank you（还有标题都叫免费白嫖了）

# 5）我收到了要求更改信息的邮件

答：信息填得不规范，按照它要求更改就行（请全英文交流）

# 6）AMD 可以吗

答：借用雷大善人的话” 这 TM 绝对是来捣乱的！“，不过可以去试试（手动狗头保命）
