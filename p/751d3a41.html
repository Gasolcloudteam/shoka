# 什么是内容分发网络（CDN）：

  1. 前言： CDN 是一种分布式网络架构，通过在全球多个服务器节点上存储网站和应用程序的静态资源，提高这些资源的访问速度和性能。
  2. 原理： 当用户请求访问网站或应用时，CDN 会自动将请求路由到最近的服务器节点，从而减少数据传输的延迟，提高加载速度。
  3. 优势：
    • 快速加载： CDN 通过使静态资源更靠近用户，减少了加载时间。
    • 降低带宽成本： 通过在全球分布数据，CDN 可以降低源服务器的负载，减少整体带宽使用。
    • 提高稳定性和可用性： 分布式节点提供了冗余，即使某个节点故障，其他节点仍然可以服务用户请求。
  4. 作用：
    • 加速网站加载速度。
    • 提供更好的流媒体服务。
    • 保护网站免受恶意攻击，如分布式拒绝服务攻击（DDoS）。
    • 改善全球用户体验，特别是对于国际访问者。
