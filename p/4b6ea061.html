# 基础命令

首先认识我们常见的 [root@localhost ~]#

root = 当前登录用户（root 是系统最高权限账户，一般是 sudo 方式临时调用）

@= 分隔符

locallhost = 主机名（英译中是本地主机）

#= 超级用户执行权限，如果是 $ 则代表权限为普通用户
