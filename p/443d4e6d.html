6种轻量级Linux桌面环境

# 前言

我们不少人应该有一些旧电脑，如果安装现在最新的 Linux 系统，很可能装不上或者运行不起来。新的系统对硬件版本都会有一定的要求。如果就想废物利用玩耍 Linux 系统，怎么办？下面汇总了几款对硬件要求极低的 Linux 发行版。

# 1. LXDE

LXDE （轻量级 X11 桌面环境）是最著名的轻量级 GUI 之一，于 2006 年首次发布，它在 Linux 和 FreeBSD 等类 Unix 平台上工作，LXDE 是许多 Linux 发行版的默认 GUI，如 Lubuntu、Knoppix、LXLE Linux、Artix 和 Peppermint Linux OS 等。

LXDE 是用 C 语言和 GTK+ 库编写的，是一个非常好的桌面环境，可以在旧计算机上运行，它是许多工具的组件，如 PCManFM（文件管理器）、LXDM（X 显示管理器）和许多其他组件。

# 在 Linux 中安装 LXDE

LXDE 可以从大多数 Linux 发行版的官方存储库中安装。

# 2. LXQT

LXQT 是 LXDE 项目中的官方 Qt 移植， LXQT 开发人员将其定义为下一代轻量级桌面环境，它是可定制的。

提供 LXQT 作为默认桌面版本的 Linux 发行版包括 Lubuntu、Fedora Linux 的 LXQt spin、Manjaro LXQt 版本、SparkyLinux LXQt，而其他发行版如 Debian 和 openSUSE 在安装过程中将其作为替代桌面环境。

# 在 Linux 中安装 LXQT

LXQT 可以从大多数 Linux 发行版的官方存储库中安装。

# 3. Xfce

Xfce 是一个用于类 Unix 平台的免费开源桌面环境，与 LXDE 不同， Xfce 不是一个非常轻量级的 GUI，但它专注于尽可能轻量级并保持漂亮的视觉外观。

Xfce 于 1996 年首次发布，它是用 C 语言编写的，带有 GTK+ 2 库，Xfce 有自己的文件管理器 Thunar ，还有许多其他组件，如 Xfwm、Xfdesktop 等。

# 在 Linux 中安装 Xfce

Xfce 可以从大多数 Linux 发行版的官方仓库中安装。

# 4. MATE

MATE 是 Gnome 2.x 期待已久的分支， MATE 能在大多数旧计算机上轻松工作，因为它是从 Gnome 2.x 分叉而来的，MATE 开发人员在 Gnome 2.x 的源代码中更改了许多内容，现在它完全支持 GTK 3 应用程序框架。

MATE 也是许多现代 Linux 发行版的默认桌面环境之一，这使其成为类 Unix 平台最受欢迎的 GUI 之一，具有直观和有吸引力的图形界面。

# 在 Linux 中安装 MATE

MATE 可以从大多数 Linux 发行版的官方存储库中安装。

# 5. Trinity Desktop

Trinity 桌面环境（TDE）是为类 Unix 操作系统创建的完整轻量级软件桌面环境，适用于喜欢传统桌面模型的个人计算机用户。TDE 诞生于 KDE 的一个分支，但现在它是一个完全独立的项目，拥有自己的开发团队。

TDE 版本提供了一个稳定且高度可定制的桌面。Trinity 被打包为 Debian、Devuan、Ubuntu、Fedora、RedHat 和其他各种发行版和架构。它也是 Q4OS 和 Exe GNU/Linux 的默认桌面环境。

# 在 Linux 中安装 TDE

Trinity 桌面可以从大多数 Linux 发行版的官方 Trinity 存储库中安装。

# 6. Create Your Own Desktop

安装轻量级桌面环境并不是拥有轻量级桌面的唯一方法，例如你还可以将任何窗口管理器与任何其他附加组件或工具一起使用，以获得好看的桌面。

  • OpenBox 对于那些喜欢简单的人来说是一个很好的窗口管理器。
  • i3 是 Linux 和 BSD 系统的轻平铺窗口管理器，可定制且文档齐全，它主要是为有经验的用户和程序员构建的。
  • FluxBox 是一个堆叠窗口管理器，最初于 2001 年从 BlackBox 分叉而来，非常简单轻量级，适用于许多平台。
  • dwm 是 X 显示服务器的动态窗口管理器，非常简单，是用 C 语言编写的。
  • 还有 JWM、PekWM、Sawfish、IceWM、FLWM 等。

还有许多其他窗口管理器。但是，除了一些有用的桌面工具外，你还可以安装任何你想要的窗口管理器，例如 Tint2（一个显示当前打开的窗口和时间的漂亮面板）、Screenlets（有名的桌面小工具程序）、Conky（一个不错的桌面系统监视器小工具）以及你可能喜欢的任何其它工具。
