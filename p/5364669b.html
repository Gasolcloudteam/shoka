# 前言

今天兴起，写一篇博文，也算是记住曾今一步步走来的脚印吧，最近记忆真的是越来越差了。

一个人到底要怎样才算是真正成长了？如果让几年前的我来回答这个问题，我肯定是毫不犹豫的说满 18 就是大人了。

后来随着年龄的增长，自己的生活变化，渐渐明白了：人一生会长大三次，第一次是在发现自己不是世界中心的时候，他会长大。这是人的第一次成长。因为，除了父母亲人没有人会让着你。你是自己父母的全部啊，却不是所有人的全部，当你遇到问题的时候你会发现，除了你的父母，没有人会帮你。

# 兴趣诞生的地方 --Flash

还记得当时是 2015 年，一次偶然，我发现了梦寐以求的软件 --flash 手机版，为什么呢？因为小时候每次玩电脑 4399 小游戏时总是能看到 flash 的经典图标，那时我便在想：如果给手机安装这个软件，我是否可以在手机上玩 4399 小游戏？

当我激动的下载好手机版 flash 时，打开手机上的浏览器，赛尔号终于可以在手机上玩了！或许是当时这个成功激起了我对于研究计算机的兴趣（后来将最开始无法运行的生死狙击也成功通过额外插件实现运行了）。2016 年（小学毕业那个假期），我开始深入对手机虚拟机的研究。

当时记得第一次用那个模拟器是 bochs，咋说呢没有达到我预期的效果，卡的要死。后来去接触了 limbo 模拟器，咋说呢效率也很低，到 2017 年，我已经将圈内的什么 APQ、QEMU 那些主流玩了个遍，各种格式的镜像效果都不行，不能流畅运行 PC QQ、浏览器。当然后来随着研究的深入我才明白，原来跨架构模拟（ARM 翻译到 X86）性能损失是很严重的，不可能完全流畅的运行。 至此，第一次对计算机的简单接触到此。

# 培养兴趣的地点 -- 云服务器

虽然开始时用了无数个夜晚研究的东西放下了，但是还是没有让我对手机 PC 化使用死心，于是，我接触到了云电脑，记得当时还是达龙云电脑一家独大，我对这种远程桌面实现手机正常电脑办公、游戏产生了极大兴趣，在用了一圈后，2018 年初二了，我正式接触云服务器，当时只会玩 Windows Server 系统的服务器，用来开 MC 服务器、挂挂 QQ……

初二下半学期，我在使用云服务时顺便将重装系统给学会了，因为当时家里已经有电脑能够经常玩到了，外加当时初中学校计算机机房也是每天中午去维护，渐渐地也加深了对计算机的一些理解。也正是因为有了电脑，所以我开始使用它运行各种虚拟机，使用腾讯云 SCF 云函数等等，这也为后来搭建网站做了很大的基础准备。还记得当时拿腾讯云 SCF 搭建了一个挂载 Onedrive 的 Onemanager。甚至当时不明白什么是域名 O (∩_∩) O 哈哈～。

玩这些东西时，顺便学了点 Python 和 Java，只是 Python 太久没用，当时学的那一点已经忘得差不多了。

# 正式开始搭建网站 -- 第一个博客

2019 年，初三上半期，经过一年多腾讯云的使用，以及对服务器、域名这些的了解，我开始尝试着手搭建网站，刚开始使用的 Z-blog 系统 + 香港虚拟主机（因为免备案便宜），但是用了没几天吧，各种问题劝退，让我开始使用 WordPress，域名嫖的 freenom 的，当时那个域名是 lihaoyun.cf 在 WordPress 里面，发布了我收集的各种破解文件、注册机以及我个人对计算机使用的技巧。

# 整理情绪，开始新的目标

搞了一年多的东西突然放弃其实还是有很多不舍，所以在 2021 年 1 月 24 日，我正式开始了本站（Hexo 站点）的搭建，顺便学了一下 Linux，这时我也已经进入职校一年了，有充足的时间让我去干想干的事，兜兜转转吧，在使用 Hexo 框架遇到了很多问题，但是每次解决都是让我对 HTML 这些一个深入的认识，总体来讲，我比以前更爱动脑了，这也使得 Linux 的学习突飞猛进。本站现在主要就是拿来给自己做学习笔记，以及一些资源分享。

# 谈谈直到现在的一些看法

其实成长并不是很完美的事情，就像一句话 *“有一天，大雄长大了，哆啦 A 梦再也拉不住大雄的手了，因为长大我们层失去太多太多！”* 虽然我现在确实感觉过的很满足，在自己喜欢的专业，做着想做的事情，但是，我很少有几个交心的朋友，现在这个班级我刚来半学期，大家都认识我，我也认识大家了，因为我很多专业知识方面超越他们，所以大家基本上也是对我避而远之，我也很少和他们说话那些，时间久了，我成了那个” 没有乐趣的人。“

依稀记得初中时，有女生送我的几句话：” 你这辈子有电脑就够了，跟你的电脑过去吧！“；“你从来没想过交个异性朋友那些吗？”……

现在回过头来看看这些事情，也明白了当时自己的天真，执着于追求技术，放下了生活，与周围格格不入。也希望大家在追求完善充实自己的过程中不要忘了同时提升自己的社交那些，我现在社交那些真的很差！

如果很多东西已经没有后悔回头的路了，也希望大家看开点，亡羊补牢 -- 为时不晚！不要总是执着于过去，过度的在意过去只会毁了我们自己！很多东西就算真的有后悔药，以我们当时的资历、认识也是注定一样的结局。做好当下的自己，过往不念，当下不杂，未来不惧！
