# win10 使用 win11 应用商店

我最近因为一些原因切换回了 win10，但是使用了半年的 win11，习惯了那个应用商店，便去找了一个方法使 win10 也能用 win11 应用商店。

# 教程部分（下载资源见下文）

  • 下载微软商店的最新版本。（获取解压后的文件）
  • 部分用户需先下载 Microsoft.UI.Xaml.2.7。（获取解压后的文件）
  • 打开 PowerShell 管理员权限。
  • 使用 cd 命令，导航到 Microsoft.UI.Xaml.2.7 文件位置，例如，如果下载的文件移动后的位置是 C 盘，输入 cd C:\

安装命令为：

  • 使用 cd 命令，导航到你保存微软商店 App 更新的位置。例如，如果下载的文件移动后的位置是 C 盘，输入 cd C:\ 。
  • 输入 Add-AppxPackage。
